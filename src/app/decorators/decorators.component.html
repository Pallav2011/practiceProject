Input Docorator
  • Input decorator is used to pass data from parent component to child component.
  • but there is one condition that is need the relationship between them.
  • @Input() Decorator is declared in Child component.
  • Syntax -> @Input() decorator_name:Datatype;

Output Docorator
  • Output decorator is used to pass data from child component to parent component with the help of EventEmitter.
  • but there is one condition that is need the relationship between them.
  • @Output() Decorator is also declared in Child component.
  • Syntax -> @Output() decorator_name : EventEmitter datatype = new EventEmitter datatype ();
