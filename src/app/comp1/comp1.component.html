Data Binding
  • It is main building block of Angular application.
  • It is communication between components and DOM.
  • data binding means we are binds data from HTML to TS and TS to HTML.
  • there are mainly two types of data binding techniques in Angular.
    1. One way Data Binding
    2. Two way Data Binding
One way Data binding
  • one way data binding means we are binding the data at only one side.
  • either HTML to TS or TS to HTML.
  • there are 3 types of one way data binding techniques
    1. String interpolation binding
    2. Property binding
    3. Event binding
String Interpolation Binding
  • we are binding string data from TS to HTML.
  • the variable value is displayed on HTML side by using {{"{{ }\}"}} double curly braces.
  • syntax -> {{ "{{Variable_name}\}" }}

  • Developer Name : {{developerName}}

  • violet coloured developer name in above line is binds by using string interpolation binding.
Property Data Binding
  • binds data from TS to HTML.
  • we are binding the properties of DOM elements by using [ ] square bracket.
  • syntax -> {{ "[Property_name] = variable_name" }}
  • following image is displayed by using property binding, we are binds the "src" property of img tag with the variable which holds the URL as a value.
Event Data Binding
  • binds data from HTML to TS by using ( ) rounded paranthesis.
  • Mostly used to trigger the events like click, hover, keyup, etc.
  • syntax -> {{ "(event_name) = method_name()" }}
  • in above syntax method_name means we can write any method in TS side and call or invoke that method using events from HTML side.
  • Image is Changed successfully !!!
  • when you click to above button then it can change the above image and display the message "image is Changed successfully" in green Colour, and its possible becouse of event binding,we are using "click" event on button and invoking the method to change the image with the help of event binding.
  • {{keyUpValue}}
  • in above textbox we are using "keyup" event, it can trigger the event on every key press. displays entered data front of textbox using string interpolation binding.
Two way Binding
  • we are binding data from two sides at a time means TS to HTML and HTML to TS.
  • it is combination of [ ] property binding and ( ) event binding.
  • syntax -> {{ "[(ngModel)]" }}
  • it is also called as "BANANA IN BOX".
  • The ngModel directive is a directive that is used to bind the values of the HTML controls (input, select, and textarea) or any custom form controls, and stores the required user value in a variable and we can use that variable whenever we require that value.
  • TextBox Value : {{value}}
  • in above textbox already value displays becouse of property binding and when you enter the new value and display that new value its by using event binding with the help of ngModel.
  • in above example we need to import FormsModule.
